# C Learning Journey 🚀

This repository contains my journey of learning the C programming language from scratch.

## 📚 Topics Covered

- Basics (Hello World, Variables, Input/Output)
- Control Flow (if-else, loops)
- Functions
- Arrays
- Pointers
- Strings
- File Handling

## 🛠️ How to Run Programs

Compile and run using GCC:

```bash
gcc filename.c -o output
./output
```

## 📁 Folder Structure

- control-flow/ → Conditions and loops

- pointers/ → Pointer concepts
- projects/ → Mini projects
